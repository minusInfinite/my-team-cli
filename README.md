## Installation

Clone the repo

```termnal
git clone https://github.com/minusInfinite/my-team-cli.git
```

Install packages via Yarn - <https://yarnpkg.com/>

```termnal
yarn install
```

## Usage

Use the following NPM script

```termnal
yarn start
```
